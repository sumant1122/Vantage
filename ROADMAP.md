# 🗺️ Shyell Roadmap

This document outlines the planned development path for Shyell. We welcome community feedback and contributions on any of these items.

## 🟢 Short Term (Next Release)
- [ ] **Job Control**: Support for background jobs (`&`, `fg`, `bg`).
- [ ] **Extended Built-ins**: Improve `history` with search capabilities.
- [ ] **Config File**: Support for a `.shyellrc` file for persistent settings.

## 🟡 Mid Term
- [ ] **Vim Mode**: Full vi-style editing support.
- [ ] **Plugin System**: Allow users to add custom modules to the dashboard.
- [ ] **Interactive Top**: A more interactive version of the `top` built-in.

## 🔴 Long Term
- [ ] **Cross-Platform Parity**: Full feature support for Windows and macOS.
- [ ] **Scripting Language**: Implement a basic shell scripting language for automation.
- [ ] **Remote Monitoring**: Ability to monitor remote servers directly in the prompt.
